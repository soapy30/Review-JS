Truthy and Falsy

조건문(if)에서
true처럼 취급되면 Truth
false처럼 취급되면 Falsy


Falsy 로 취급되는 값 목록

  • false
  • 0
  • ""
  • null
  • undefined
  • NaN
주의사항
  1. prompt()는 string 또는 null만 반환
  2. undefined는 입력 없음이 아니라 변수 상태
  3. "" vs " "는 완전히 다름. 공백이라도 있다면 TRUTHY / 빈문자열 FALSY
  4. 0 / NaN은 Number 변환 결과
  5. Falsy는 의도적으로 체크해야 구분 가능
