메서드

메서드 - 객체 내부에 정의된(객체의 프로퍼티로 저장된) 함수로,
해당 객체가 할 수 있는 행동을 의미함

"."를 통해 접근하거나 ["메서드명"]으로 접근

함수는 어디에 저장되었느냐 따라 메서드가 될 수 있을뿐, 본질은 함수다.(이후 this에서 후술)

this keyword

this는 함수가 “어디서 정의됐는지”가 아니라 “어떻게 호출됐는지”로 결정된다
