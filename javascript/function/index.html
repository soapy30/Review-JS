Function

기본 함수

함수정의 -> 함수호출 순서로 사용해야 한다

function 함수명 (){실행내용} 의 형태로 정의, 호출시 함수명() 으로 호출하여 실행

연관 개념= hoisting : 변수와 함수 선언이 실행 전에 먼저 처리되는 것처럼 보이는 현상

Arguments: 인수 함수

늘 같은값 출력하지 않고, 전달받은 인자에 따라 다른 값을 출력할 수 있다

function 함수명(인자){실행내용}으로 정의하고, 호출시 함수명(인자)로 호출하여 실행

function greet(name){console.log(`Hi, ${name}!`)}

위 코드에서, name은 매개변수(parameter)로, 전달된 값을 출력한다
