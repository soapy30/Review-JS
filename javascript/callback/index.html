함수를 인자로 사용하는 메서드들

다른 함수(또는 메서드)에 인자로 전달되어, 나중에 호출되는 함수를 콜백 함수(callback function)라고 한다.

참고: 개념 흐름
1. 함수도 하나의 값이다.
2. 따라서 함수도 인자로서 전달 가능하다.
3. 그 인자로 전달된 함수는, 바로 실행하지 않고
4. 필요한 시점에(나중에:callback) 호출된다.

  • ForEach - 각 요소로 어떤 행동을 함

    • forEach는 for..of 나오기 전에 많이 사용

    반환값은 항상 undefined

  • Map - 바꾸기(변환)

    • 각 요소를 다른 값으로 바꿔서 새 배열 반환. 원본 건들지 않음

    콜백함수의 return값이 새 배열의 요소가 됨

    실무에서 가장 많이 사용

  • Filter - 골라내기

    • 조건에 맞는 요소만 골라서, 새 배열을 반환함


요약

단순 반복 + break/continue 필요 → for…of
배열 메서드 체이닝 → map / filter/ reduce
단순 출력 → forEach or for…of 둘 다 가능
